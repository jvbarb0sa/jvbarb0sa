## Operating Standard

<table>
  <tr>
    <td width="25%" align="center" valign="top">
      <h3>Clarity</h3>
      <p>Every screen needs hierarchy, purpose and an obvious next action.</p>
    </td>
    <td width="25%" align="center" valign="top">
      <h3>Precision</h3>
      <p>Spacing, typography, motion and interaction must feel intentional.</p>
    </td>
    <td width="25%" align="center" valign="top">
      <h3>Performance</h3>
      <p>Premium products must feel fast, responsive and technically stable.</p>
    </td>
    <td width="25%" align="center" valign="top">
      <h3>Business Logic</h3>
      <p>Beautiful interfaces still need to solve real operational problems.</p>
    </td>
  </tr>
</table>
